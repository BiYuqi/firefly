# firefly

> A simple online chat platform based on nodeJs mongodb express and vueJs

# Usage
```js
// 前台
git clone https://github.com/BiYuqi/firefly.git

cd firefly

npm i

npm run dev

// 后台
npm i supervisor -g // 如果已经安装过，此步骤可省略

npm run app
```
